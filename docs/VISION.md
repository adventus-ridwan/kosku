# Kos Map - Vision

## Project Vision

Kos Map is an interactive boarding house management application centered around a visual floor plan.

Instead of managing rooms from tables, users manage the boarding house directly from an interactive map.

The map is the primary interface of the application.

---

# Target Users

* Boarding House Owner
* Boarding House Manager (Penjaga)
* Prospective Tenant (Public)

---

# Primary Goals

Build a portfolio-quality SaaS application that demonstrates:

* Interactive map editing
* Business-oriented data modeling
* Role-Based Access Control (RBAC)
* Clean Architecture
* Scalable code organization

The project is intentionally **not** an enterprise system.

Version 1 focuses on simplicity, maintainability, and user experience.

---

# Product Philosophy

Map First.

Everything starts from the map.

Examples

Click Room

↓

View information

↓

Manage tenant

↓

View contract

↓

View history

The user should rarely need to leave the map.

---

# Design Philosophy

* Simple
* Fast
* Clean
* Professional

Avoid unnecessary complexity.

Every feature should solve a real business problem.

---

# Version 1 Scope

Version 1 includes:

* Interactive Room Map
* Facility Management
* Role System
* Tenant Management
* Contract History
* Dashboard
* Public Information
* Deployment

Everything else is considered future development.

---

# Long-Term Vision

Future versions may include:

* Payment tracking
* Invoice generation
* Online booking
* Multi-property support
* Analytics
* Mobile application

These are intentionally excluded from Version 1.
